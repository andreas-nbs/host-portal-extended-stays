# Host Portal Extended Stays

Standalone HTML prototype for the VLOO host portal extended-stay request flow.

## Files

- `index.html` - host portal extended-stay request prototype.

## Notes

- Incoming extended-stay requests show user, company, request details, and potential earnings.
- Hosts can add the request to the flexible calendar preview, review updated capacity, then approve or reject.
- The prototype is static HTML/CSS/JavaScript and can be opened directly in a browser.
